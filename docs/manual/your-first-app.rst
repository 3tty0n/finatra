Your first app
==================

The easiest way to get going with finatra, is to install the finatra gem.

.. code-block:: guess

   gem install finatra


Then generate a new app using

.. code-block:: guess

   fin new MyApp

Now you should be able go into MyApp and start the app

.. code-block:: guess

   cd MyApp
   fin start

Make sure it's running on http://localhost:7070
